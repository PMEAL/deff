See @README.md for project overview

- This project uses the `uv` package manager. If you need to run any python commands, be sure to activate the virtual environment first with `source .venv/bin/activate`.
- To install package X use `uv pip install X`.
- To include package X in the project's dependencies use `uv add X`.
- All docstrings should be written in the numpydoc format (not napolean or google style)
